This app requires a web3 capable browser i.e, Mist, Metamask, etc. It uses web3 to interface with the Ethereum network.

Currently, this app is under active development so please do not use it with your Ethereum Mainnet address. We suggest using Rinkeby test network.

You can get Rinkeby Ethereum for free/ small effort from one of its faucets. Then you can use them to play around with this app.
